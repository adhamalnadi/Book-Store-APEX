A web-based library management system built using Oracle APEX and Oracle Database.

This application allows libraries to manage books, members, and borrowing operations through a simple and secure interface. It supports book catalog management, user registration, loan tracking, and administrative control with authentication.

Developed as a graduation project to demonstrate database design, PL/SQL programming, and real-world system implementation using Oracle APEX.
